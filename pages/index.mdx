# Introduction

Varlet is a web development environment for minimalists, inspired from Laravel Valet and Laragon. Varlet only includes
PHP, Composer, and HTTP web server. If you want to use databases, you need to install them separately.

## Database as add-on

The Varlet package does not include database engines such as MariaDB / MySQL, PostgreSQL, or even Redis. You are free to
install any database distribution as you wish. Or you can use our simplified database distribution at Varlet Addons.
